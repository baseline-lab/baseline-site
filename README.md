# baseline-lab.com

Static marketing + legal site for Baseline Labs LLC and the Halo app.
No build step — plain HTML/CSS. Hosted on GitHub Pages at https://baseline-lab.com.

- `index.html` — company home + Halo product
- `privacy.html` — Halo privacy policy (App Store + Google OAuth URL)
- `terms.html` — Halo terms of service
- `styles.css` — shared styles
- `CNAME` — custom domain for GitHub Pages
